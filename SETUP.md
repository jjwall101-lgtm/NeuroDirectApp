# NeuroDirect v26 setup

## Simple setup

1. Download and extract the ZIP.
2. Upload every file directly into GitHub root.
3. Commit changes.
4. Open the app links below.

Teen:
https://jjwall101-lgtm.github.io/teen.html?v=26

Parent:
https://jjwall101-lgtm.github.io/parent.html?v=26

## What to check

1. Open Teen on your phone.
2. Tap the menu.
3. The left menu should stay on the left.
4. The preview/cards should stay in a narrow right-side column.
5. Nothing should drop underneath.
6. Repeat on Parent.
