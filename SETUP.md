# NeuroDirect v16 setup

## Simple setup

1. Upload every file in this package to GitHub.
2. Commit changes.
3. Publish firestore.rules in Firebase.
4. Open teen.html.
5. Teen signs in and creates/saves family code.
6. Open parent.html.
7. Parent signs in and enters same family code.
