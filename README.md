# NeuroDirect

NeuroDirect is a simple mobile-first family step tracking web app.

## What is included

- `index.html` — main app page
- `style.css` — full app styling
- `script.js` — app logic and local saving
- `manifest.json` — installable PWA setup
- `sw.js` — offline cache/service worker
- `icon-192.png` — app icon
- `icon-512.png` — app icon
- `UPLOAD_NOTES.txt` — simple GitHub upload steps

## Features

- Add family members
- Parent, Kid, or Teen role
- Daily step goals
- Manual step entry
- Quick-add step buttons
- Leaderboard
- 7-day history
- Daily rollover
- Local-only storage
- Installable as a web app

## Important

This version saves data locally on the device using browser local storage. It does not sync between devices.
