This package replaces the old logo with the selected premium ND neural logo.

Upload steps:
1. Delete every old file from the GitHub repo root.
2. Upload every file from this package into the repo root.
3. Make sure these files are included:
   - index.html
   - style.css
   - script.js
   - manifest.json
   - sw.js
   - logo.svg
   - logo.png
   - icon-192.png
   - icon-512.png
   - README.md
   - .nojekyll
4. GitHub Pages should be set to main branch / root.
5. Wait for GitHub Pages to rebuild, then refresh Chrome.
