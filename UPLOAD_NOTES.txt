NeuroDirect v10 upload notes — FIREBASE USER PROFILES

1. Delete the old repo files or overwrite them with this package.
2. Upload every file from this ZIP into the repo root.
3. Open firebase-config.js.
4. Paste your Firebase web app config.
5. Change firebaseEnabled from false to true.
6. Enable Email/Password Authentication in Firebase.
7. Create Firestore Database.
8. Add the rules from firestore.rules.
9. Open the app, go to Settings, and create a profile.
