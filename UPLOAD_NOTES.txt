NEURODIRECT UPLOAD NOTES

Upload these files into the ROOT of your GitHub repository:

index.html
style.css
script.js
manifest.json
sw.js
logo.svg
icon-192.png
icon-512.png
README.md
UPLOAD_NOTES.txt

Do not put them inside an assessment folder or any other folder.

Default parent PIN: 1234

After GitHub Pages is enabled, open the published link on the phone.
On Android/Chrome, use:
Menu → Add to Home screen → Install

If the app does not update after you upload changes:
1. Open the app link in Chrome.
2. Refresh it a few times.
3. If needed, clear site data for that GitHub Pages site.
4. Future versions can also update the cache name inside sw.js.
